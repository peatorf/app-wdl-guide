# app-wdl-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为通用存档与展示用途。

## 项目简介

这是一个轻量级的 HTML 页面集合仓库。每个页面均为独立文件，可直接通过浏览器查看。仓库本身不涉及后端服务、数据库或动态内容生成，所有页面均为静态 HTML。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页（如有）
├── pages/              # 存放主要 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、CSS、JS 等）
│   ├── styles/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各独立 HTML 文件，文件命名建议使用英文小写与连字符。
- `assets/` 目录用于存放页面引用的静态资源，可按类型进一步细分。

## 页面归档说明

- 所有页面均为独立、自包含的 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无强依赖关系，可单独使用或组合展示。
- 归档内容以通用信息展示为主，不涉及具体商业、营销或推广信息。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或对现有页面进行修正。
- 新增页面请放入 `pages/` 目录，并确保资源文件路径正确。
- 请保持页面内容简洁、通用，避免包含特定域名或网址。
- 本仓库不提供在线演示或托管服务，仅作为源码归档。
